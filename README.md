# Mini Katalog Uygulaması

## 📝 Açıklama
Flutter ile geliştirilmiş, kullanıcıların ürünleri pratik bir şekilde listeleyebildiği ve detaylarını görüntüleyebildiği mini bir katalog uygulamasıdır.

## 🛠 Kullanılan Teknolojiler ve Sürümler
* **Flutter:** Son Sürüm
* **Dart:** Son Sürüm

## 🚀 Çalıştırma Adımları

Projeyi kendi ortamınızda çalıştırmak için terminalinizde sırasıyla aşağıdaki adımları izleyin:

1. **Projeyi Klonlayın:**
   ```bash
   git clone <github-repo-linkinizi-buraya-yapiştirin>
'''
 2. **Proje Klasörüne Geçiş Yapın:**
   ```bash
   cd mini_katalog_uygulamasi
'''
3.**Uygulamayı Çalıştırın:**
''' bash
  flutter pub get
'''
4.**Uygulamayı Çalıştırın:**
''' bash
  flutter run
'''
